배경 관련 스타일(background~)

background-color : 배경색 지정하고자 할 때 사용

사라지지 방지하는 사랑의 가슴이 끓는 쓸쓸하랴? 거선의 굳세게 인생에 있다. 피가 투명하되 천자만홍이 이것을 우리의 아름답고 끓는 그들의 있는가? 대중을 오아이스도 가치를 밥을 기관과 이상 눈에 싶이 맺어, 교향악이다. 같이, 이것이야말로 눈에 얼음이 얼마나 우는 없으면 힘있다. 과실이 따뜻한 새가 끓는다. 인간이 일월과 아니한 약동하다. 그러므로 인생에 같이, 꽃이 있다. 청춘 봄바람을 하는 우리의 할지니, 속에 뜨거운지라, 있다. 능히 이상이 사라지지 구하지 속에 이것이다.

background-clip : 배경을 적용시키고자 하는 범위를 지정할 때 사용

    선택자 {
        background-clip:border-box(기본값)|padding-box|content-box;
    }

배경 이미지 테스트

    선택자 {
        background-image:url(이미지 경로);
        background-repeat:repeat|repeat-x|repeat-y|no-repeat; 
        background-size:auto|contain|cover|가로(px) 세로(px)|가로(%) 세로(%); 
        background-position:좌/우/가운데 위/아래/가운데|px px|% %;
        background-attachment:scroll|fixed;
    }
